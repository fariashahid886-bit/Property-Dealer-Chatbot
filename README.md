# Conversational-Agent

A Python-powered conversational interface designed to model the profile of Muhammad Shahid. This project focuses on human-centric data mapping using conditional logic and keyword-based response routing.

## 📋 Logical Architecture
The application is structured as a non-terminating state machine, utilizing a `while True` loop to manage the conversation flow. User inputs are normalized through string conversion and mapped against a decision tree composed of `if-elif-else` control blocks.

### Key Logic Modules:
* **Identity Management:** Biographical data retrieval (Name, Age, Location).
* **Behavioral Heuristics:** Processing queries related to personality traits, temperament, and social preferences.
* **Routine Mapping:** Dynamic tracking of daily habits, work ethics, and responses to stressors.
* **Fallback Strategy:** A robust `else` block serves as a navigational safety net to handle undefined user intents.
